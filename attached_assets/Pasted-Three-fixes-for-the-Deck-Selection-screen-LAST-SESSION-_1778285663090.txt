Three fixes for the Deck Selection screen "LAST SESSION" card:

# FIX 1 — Show full deck name
The card currently shows partial deck names. It must show the FULL display name 
based on deckId from verba_last_session in localStorage.

Mapping (deckId → display name):
- "essential" → "Essential English"
- "advanced" → "Advanced English"
- "gre" → "GRE Vocabulary"
- "myverba" → "My Verba"

If the saved deckId in localStorage is "gre", the card MUST show "GRE 
Vocabulary", not "Vocabulary" or any partial version.

# FIX 2 — Show difficulty after deck name
The card must show the difficulty next to the deck name, separated by " · ":

Display format: "<icon> <Full deck name> · <Difficulty capitalized>"

Examples:
- "<icon> GRE Vocabulary · Hard"
- "<icon> Essential English · Easy"
- "<icon> Advanced English · Medium"

Exception: if deck is "myverba", show only "<icon> My Verba" (no difficulty 
because My Verba doesn't have one).

The difficulty value comes from verba_last_session.difficulty in localStorage.
Capitalize the first letter: "easy" → "Easy", "medium" → "Medium", "hard" → "Hard".

them everywhere they appear with Lucide icons.

Install lucide-react if not already installed.

Required Lucide icons per deck:
- "essential" → <BookOpen /> 
- "advanced" → <Library />
