# PROMPT 3 — Verba Results Screen (Step 5)

Build the **Results Screen** for Verba. This screen appears at the end of every quiz session, replacing the current placeholder behavior of returning to the Pre-Quiz screen.

Maintain the exact same visual style as the existing screens: dark background #0A0A0A, amber accents #D97706 / #F59E0B, sage green #84A98C for antonyms, Space Grotesk for headings, Inter for body, soft amber radial halos in background, generous spacing. Floating GRE words in the background should be present but at 30% opacity (less than welcome screen) and only in safe zones (corners, edges) — they must NOT overlap with central content.

Use Framer Motion for all animations.

## LAYOUT — Single vertical scroll, top to bottom

### SECTION 1 — Hero score block (top of screen, full viewport height on first load)

- Top of section: a small motivational title that varies by performance
  - score >= 90% → "Outstanding!"
  - score 70-89% → "Great work"
  - score 50-69% → "Keep going"
  - score < 50% → "Every word counts"
  - Style: Inter weight 500, font-size 18px, letter-spacing 0.05em, color amber #D97706

- Center: huge score display
  - Format: "8 / 10" (correct out of total)
  - Style: Space Grotesk weight 700, font-size 96px, white-to-amber gradient (same shimmer as Verba title)
  - Animation on entry: numbers count up from 0 to final value over 1.5 seconds, easing out

- Below score: percentage in smaller text
  - "80% accuracy" — Inter weight 400, font-size 24px, color rgba(255,255,255,0.7)

- **Perfect session celebration:** if score === total (100%), add a subtle confetti burst animation that plays once on entry. Use small amber, white, and sage green particles that drift upward and fade. Background of this section gets a subtle rainbow gradient overlay at 10% opacity for 3 seconds, then fades to normal.
